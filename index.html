01. Extention Installation

01. Extention has a link structure to 'VFXPORT' folder, so make sure you Don't change files structure or names!

02. Put 'VFXPORT' folder preferably in root of your drives.

Make sure there is:
1) No spaces
2) No foreign or special characters in 'VFXPORT' path

good   C:/MyFiles/VFXPORT
good   E:/VFXPORT

bad      C:\Мои Документы\VFXPORT
bad      C:/My Files/VFXPORT
